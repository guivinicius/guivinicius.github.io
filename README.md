# guivinicius.github.io

Personal website of Guilherme Vinicius Moreira - Software Engineer based in Vancouver, Canada.

## About

Brazilian software engineer living in Vancouver. Passionate about building things, exploring new technologies, and finding ways to help others along the way.

Former specialty coffee shop and roastery owner (10 years in Brazil). Now focused on software development and creating tools that solve real problems.
